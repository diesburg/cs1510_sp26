CS 1510 Python Information

Installing Python/IDLE on your computer

Python is free to install. If you have a PC or Mac and the ability to install programs, please install the newest version of Python from the Python website. If you have Linux, come see me for the package names.

If you are renting a PC from UNI and cannot find IDLE, please type "Software Center" in the search bar to run Software Center. (Software Center allows you to install approved UNI software.) Click on the Python icon under Applications to install Python and IDLE.

Chromebooks and other situations

You can use UNI's virtual desktop (Citrix) option to run Python/IDLE. 

Alternatively, if you have a chromebook and are comfortable using the command line, you can install Python and IDLE using the underlying Linux operating system. Here is an article that explains how.
